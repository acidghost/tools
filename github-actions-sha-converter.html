GitHub Actions Tag → SHA Converter

Convert GitHub Actions references from tagged format to commit SHA format for reproducibility. Optionally expands to full version tags.

Example formats: actions/checkout@v4 (expands to v4.2.2), actions/checkout@v4.2 (expands to v4.2.2), or actions/checkout@v4.1.0 (exact)

Without token: 60 requests/hour. With token: 5000 requests/hour. Stored locally in your browser.

Built with vanilla HTML/JS. No build step required.
